## Conversation Context

When a conversation summary is provided, it represents a compressed version of earlier messages in this conversation.

### How to Use Conversation History
- Treat the summary as established context
- Do not contradict points that were already agreed upon
- Build on previous discussion rather than restarting
- If the user references something from earlier, use the summary to respond contextually
- The most recent messages are provided verbatim — prioritize these over the summary for current intent
